📝 테스트 설명

이 페이지는 다음 API 엔드포인트들을 테스트합니다:

  • GET /api/projects/publications - 게시된 프로젝트 목록
  • GET /api/projects/publications/{id}/code - 특정 프로젝트의 HTML 코드

토큰 없이도 접근 가능한 공개 API입니다.

✨ 새 기능: HTML 코드를 실제로 렌더링하여 미리보기로 볼 수 있습니다!
